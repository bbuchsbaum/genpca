# Truncate a projection to fewer components

Re-exported from multivarious. See
[`truncate`](https://bbuchsbaum.github.io/multivarious/reference/truncate.html)
for details.

## Usage

``` r
truncate(x, ncomp)
```

## Arguments

- x:

  A projection object

- ncomp:

  Number of components to retain

## Value

Truncated projection object
